# Папка для изображений галереи

Добавляйте свои фотографии работ салона в эту папку и используйте их в портфолио.

## Рекомендации

- **Формат**: JPG (для фото) или PNG (если нужна прозрачность).
- **Размер**: 1200×900 px (или 4:3) — оптимален для галереи и lightbox.
- **Вес**: сжимайте изображения до 300–500 KB для быстрой загрузки.
- **Наименование**: используйте понятные названия:
  - `haircut-1.jpg`, `haircut-2.jpg` — для стрижек
  - `manicure-1.jpg`, `manicure-2.jpg` — для маникюра
  - `spa-1.jpg`, `spa-2.jpg` — для SPA
  - `brows-1.jpg` — для коррекции бровей
  - `lashes-1.jpg` — для ресниц
  - `makeup-1.jpg` — для макияжа

## Как добавить изображение в портфолио

Отредактируйте `portfolio.html` и добавьте строку вида:

```html
<div class="portfolio-item reveal" data-category="manicure">
  <a class="thumb" href="images/manicure-3.jpg" data-index="N">
    <img src="images/manicure-3.jpg" alt="Маникюр 3">
  </a>
</div>
```

Замените:
- `images/manicure-3.jpg` — путь к локальному изображению
- `manicure` — категория (haircut, manicure, spa, brows, lashes, makeup)
- `N` — номер индекса (1, 2, 3, ...)
- `Маникюр 3` — описание работы

Если файл не найдётся, сайт автоматически загрузит картинку с Unsplash (fallback).

## Категории

- `haircut` — стрижки
- `manicure` — маникюр и педикюр
- `spa` — SPA и массаж
- `brows` — коррекция бровей
- `lashes` — наращивание ресниц
- `makeup` — макияж и коррекция окрашивания

---

Если у вас есть вопросы, проверьте комментарии в `portfolio.html`.
